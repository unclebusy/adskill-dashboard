# AdSkill Dashboard

Offers management dashboard with advertising campaign analytics

## Tech Stack

- **React 18** + **TypeScript** with strict typing
- **Redux Toolkit** for state management
- **Vite** for bundling
- Custom UI components without third-party libraries

## Features

- Sorting by any column on header click
- Row selection via checkboxes
- Responsive table with mobile card view
- Expandable rows with detailed offer information
- Loading and error state handling

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Type check
npx tsc --noEmit
```
